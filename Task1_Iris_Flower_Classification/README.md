# Code_Alpha_Iris Flower Classification


## Description
This task involves classifying Iris flowers into three species:
Setosa, Versicolor, and Virginica using machine learning techniques.

## Dataset
- Iris flower dataset
- Sepal length
- Sepal width
- Petal length
- Petal width

## Libraries Used
- Python
- Pandas
- Scikit-learn

## Model Used
- Classification model

## Evaluation
- Accuracy score
- Classification report

## Outcome
The model successfully classifies Iris flower species with good accuracy.
